# Directive: 카드뉴스 디자인 스펙

> 출처: @indoorsy_squirrel  
> "몇 가지 규칙만 지키면 깔끔하면서도 가독성 좋은 카드뉴스"

## 기본 규격
- **사이즈**: 1080 × 1350 px (인스타그램 세로 기준)
- **포맷**: PNG (기본) / **GIF 권장** (움직이는 이미지가 집중도 훨씬 높음)

---

## 타이틀 페이지 규칙

**정렬**: 좌측 정렬로 가독성을 높인다  
**폰트 추천**: 프리텐다드, 에스코어드림 (깔끔한 느낌)

| 텍스트 요소 | 크기 | 굵기 |
|------------|------|------|
| 타이틀 | **60p** | 두껍게 |
| 부제목 | **45p** | 보통 |
| 출처 | **25p** | 두껍게 |
| 콘텐츠 설명 | **25p** | 얇게 |

---

## 본문 페이지 규칙

- **사진을 텍스트보다 크게** 넣어 한 눈에 알아보기 쉽게 만든다
- **내용은 3줄 이상 넘기지 않는다**

| 텍스트 요소 | 크기 | 굵기 |
|------------|------|------|
| 출처 | **15p** | 얇게 |
| 본문 타이틀 | **45p** | 두껍게 |
| 본문 내용 | **28p** | 얇게 |

---

## 미리캔버스 제작 가이드 (STEP별)

### STEP.01 — 사이즈 선택
- 상단 사이즈 탭 클릭 → **1080 × 1350px** 설정 후 적용
- (또는 직접 입력: 너비 1080, 높이 1350, 단위 px)

### STEP.03 — 그라데이션 넣기
- 텍스트 가독성을 높이기 위해 이미지 하단에 **투명 그라데이션**을 배치한다
- 요소 메뉴 → "투명 그라데이션" 검색 → 검정 그라데이션 선택 → 이미지 하단 배치
- 텍스트가 훨씬 더 잘 보인다

### STEP.04 — 텍스트 입력 (미리캔버스 기준 권장 크기)
- 왼쪽 '제목 텍스트 추가' 버튼 → 제목 입력
- '부제목 텍스트 추가'로 보조 설명 추가
- 텍스트 색상을 **흰색**으로 변경

| 텍스트 요소 | 크기 (미리캔버스 기준) |
|------------|----------------------|
| 제목 텍스트 | **67p** |
| 부제목 텍스트 | **38p** |

> 📌 @indoorsy_squirrel 기준(타이틀 60p/부제목 45p)과 약간 다름. 두 수치 모두 유효하며 콘텐츠에 맞게 선택

---

## 그라데이션 활용 (from @miricanvas)

- 텍스트 가독성을 높이기 위해 이미지 하단에 **투명 그라데이션**을 배치한다
- 미리캔버스: 요소 → "투명 그라데이션" 검색 → 검정 그라데이션 선택 → 이미지 하단에 배치

---

## GIF 활용 팁

- 멈춘 이미지보다 **움직이는 이미지가 집중도가 훨씬 높다**
- PNG보다는 영상을 넣어 **GIF로 저장**하면 좋다
- 인스타그램에서 GIF는 자동재생되어 피드에서 눈에 띈다

---

## AI HTML 생성에 적용할 규칙

```text

- font-weight: 900 (타이틀), 700 (출처), 300 (설명)

- padding: 80px

[본문 슬라이드 CSS 기준]
- font-size: 45px (섹션타이틀), 28px (본문)
- font-weight: 700 (섹션타이틀), 300 (본문)
- 텍스트는 3줄 이내
- 이미지 영역을 텍스트보다 크게 배치 (CSS div로 표현)

[공통]
- 폰트: Noto Sans KR (웹 한글 폰트)
- overflow: hidden (각 슬라이드)
- 투명 그라데이션: background: linear-gradient(to top, rgba(0,0,0,0.7), transparent)
```
